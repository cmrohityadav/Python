## Virtual Environment

### Create

```bash
python -m venv myenv

# Run on linux
source myenv/bin/activate

# Run on windows
.\myenv\Scripts\activate


# To deactivate
deactivate

```
### Install dependencies
pip install -r requirements.txt
pip list
